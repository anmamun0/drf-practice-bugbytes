`drf-spectacular` is a powerful library for auto-generating OpenAPI documentation from your Django REST Framework (DRF) project.  It integrates easily with Swagger UI, ReDoc, and Rapidoc for interactive API exploration.


### Tool	Description
- `drf-spectacular` : 	Auto-generates OpenAPI documentation for DRF
- `Swagger UI`	Interactive browser-based API documentation
- `ReDoc	`Modern, clean documentation viewer
- `Schema JSON`	Raw OpenAPI 3 schema export
